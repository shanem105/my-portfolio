Wing-Wide Network Infrastructure Upgrade

SCOPE: Physical Layer, VLAN Config, Enterprise Switching

1. The Mission

I participated in a critical base-wide infrastructure refresh, replacing aging switch hardware, installing UPS devices, and PDUs to support increased data throughput and improve network reliability for mission-critical operations.

Figure 1: Logical Topology demonstrating VLAN segmentation between Voice and Data traffic.


2. Implementation Steps

  • Physical Installation: Racking and stacking new enterprise switches, managing cable runs for airflow and organization.
  • VLAN Configuration: (Note: Configuration values have been sanitized for OPSEC) Configured 802.1Q tagging to segregate Data (VLAN 10) and Voice (VLAN 20) traffic, ensuring QoS for VoIP phones.
  • Documentation: Observed creation of logical network diagrams (see Fig 1 ex.) to map IP schemas and port assignments for future troubleshooting.
  • Validation: Conducting post-installation connectivity tests to ensure zero downtime for critical assets.
  • Sanitization: Managed the Sanitization of legacy switches for recycling them to other units in need of the equipment.

3. Operational Impact

This upgrade reduced network latency and eliminated massive packet loss issues associated with the legacy hardware. It also standardized the configuration across the base, making future troubleshooting significantly faster.



Result: Enhanced operational readiness and secured the physical layer of the base network.
